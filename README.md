# Mpox
关于猴痘论文的代码
先运行seir_network_simulation_new.py文件，再运行其它的方法文件。
其中该文件运行时需要根据不同国家更改一些参数，给出我们的参考参数如下：
DRC:E0 = 4.99e-4, I0 = 2.71e-5, beta = 0.376;
Uganda: E0 = 1.06e-4, I0 = 6.03e-4, beta=0.415;
Kenya: E0 = 8.13e-7, I0 = 3.44e-8, beta=0.466;
Zambia: E0 = 2.96e-6, I0 = 3.95e-6, beta=0.524;

其中还可对程序文件中的一些其它参数进行修改，可能会导致得到的结果与文献有出入，如网络节点数、选取的参考时间点等。
